# 1 - Guest
- register
- login
- view public trips
- view homepage
# 2 - Trip Member
- can join a trip
- votes on activities
- proposes a activity
- opt-in or out of a specific activity
- add an expense 
- can "Sign-off" on the final balance
- can chat and comment on specific activity
- can vote on sensitive topics (budget limits)
- can view their own respective documents
- can upload their own documents
# 3 - Trip Leader
- creates a trip
- confirms/rejects activities proposed
- sets the trip's budget threshold
- has a tie-breaking vote
- can chat and comment (generalization)
- can give "editor" permissions to other members
- can view sensitive documents of members
# 4 - Admin
- manage users (add, edit, delete) CRUD
- manages trips (view, delete)
- monitors the system
# 5 - System
- generates daily briefing
- budget threshold alert
- close expired polls
